@startuml
actor GoogleCalendar as "Google Calendar"
participant GoogleCalendarTrigger as "Google Calendar Trigger"
participant AIAgent as "Agente de IA"
participant WhatsAppAPI as "WhatsApp API"
participant TelegramAPI as "Telegram API"

GoogleCalendar -> GoogleCalendarTrigger: Notificação sobre término de evento
GoogleCalendarTrigger -> AIAgent: Processa resumo do evento concluído
AIAgent -> AIAgent: Gera resumo detalhado (ex.: título, duração, tarefas pendentes)
AIAgent -> WhatsAppAPI: Envia resumo final pelo WhatsApp
AIAgent -> TelegramAPI: Envia resumo final pelo Telegram
@enduml

@startuml
actor GoogleCalendar as "Google Calendar"
participant GoogleCalendarTrigger as "Google Calendar Trigger"
participant AIAgent as "Agente de IA"
participant WhatsAppAPI as "WhatsApp API"
participant TelegramAPI as "Telegram API"

GoogleCalendar -> GoogleCalendarTrigger: Notificação sobre criação/início de evento
GoogleCalendarTrigger -> AIAgent: Processa informações do evento
alt Dados incompletos
    AIAgent -> AIAgent: Preenche informações padrão (ex.: data de hoje, título genérico)
else Dados completos
    AIAgent -> AIAgent: Confirma validade das informações
end
AIAgent -> WhatsAppAPI: Notifica usuários sobre o evento pelo WhatsApp
AIAgent -> TelegramAPI: Notifica usuários sobre o evento pelo Telegram
@enduml
